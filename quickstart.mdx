## Sign up for an Unleash Enterprise Cloud trial

Start by signing up for a [free 14-day trial](https://www.getunleash.io/plans/enterprise-payg) of Unleash Enterprise Cloud. After you submit the form, you will receive a confirmation email. Follow the link in the email to set your password and log in to your Unleash instance.

## Create your first feature flag

Once you've logged in, it's time to create your first feature flag:
<Steps>
<Step>
 In the Unleash Admin UI, open the **Default** project.
</Step>
<Step>
Click **New feature flag**.
</Step>
<Step>
Enter a name, and click **Create feature flag**.
</Step>
</Steps>
For more details on creating feature flags, see [How to create a feature flag](https://docs.getunleash.io/how-to-create-feature-flag).

## Connect your application to Unleash

Next, use one of the client or backend [SDKs](https://docs.getunleash.io/reference/sdks) to connect Unleash with your application.


<Accordion title="Need an example application to test with?">
Explore our SDK examples directly in your browser with CodeSandbox. Check out the Unleash SDK Examples repository to try one now.
</Accordion>


<Tabs>
<Tab title="Connect a frontend SDK">
<Steps>
<Step title="Create an API token">
	In the Unleash Admin UI, create a [frontend API token](https://docs.getunleash.io/reference/api-tokens-and-client-keys#frontend-tokens) in **Admin settings > Access control > API access**.
</Step>
<Step title="Get your Unleash API URL">

	Find the base URL of your Unleash Cloud instance (for example, `https://us.app.unleash-hosted.com/some-instance-id`). Your frontend API URL is this base URL with `/api/frontend` appended to it.
</Step>
<Step title="Initialize the SDK">
	In your application code, initialize the SDK using your API URL and token.
</Step>
</Steps>
The following example shows how to use the [JavaScript SDK](https://docs.getunleash.io/reference/sdks/javascript-browser) to connect to your Unleash instance:

```javascript
import { UnleashClient } from "unleash-proxy-client";

const unleash = new UnleashClient({
    url: "https://<YOUR_UNLEASH_INSTANCE>/api/frontend",
    clientKey: "<YOUR_TOKEN>",
    appName: "<YOUR_APP_NAME>",
});

unleash.on("synchronized", () => {
    // Unleash is ready to serve updated feature flags.
    // Check a feature flag
    if (unleash.isEnabled("some-flag")) {
        // do cool new things when the flag is enabled
    }
});
```
</Tab>

<Tab title="Connect a backend SDK">
<Steps>
<Step title="Create an API token">
	In the Unleash Admin UI, create a [backend token](https://docs.getunleash.io/reference/api-tokens-and-client-keys#backend-tokens) in **Admin settings > Access control > API access**.
</Step>
<Step title="Get your Unleash API URL">
	Find the base URL of your Unleash Cloud instance (for example, `https://us.app.unleash-hosted.com/some-instance-id`). Your API URL is this base URL with `/api` appended to it.
</Step>
<Step title="Initialize the SDK">
	In your application code, initialize the SDK using your API URL and token.
</Step>
</Steps>
The following example shows how to use the [Node.js SDK](https://docs.getunleash.io/reference/sdks/node) to connect to your Unleash instance:

```javascript
const { initialize } = require("unleash-client");

const unleash = initialize({
    url: "https://<YOUR_UNLEASH_INSTANCE>/api",
    appName: "<YOUR_APP_NAME>",
    customHeaders: {
        Authorization: "<YOUR_TOKEN>",
    },
});

unleash.on("synchronized", () => {
    // Unleash is ready to serve updated feature flags.
    if (unleash.isEnabled("some-flag")) {
        // do cool new things when the flag is enabled
    }
});
```
